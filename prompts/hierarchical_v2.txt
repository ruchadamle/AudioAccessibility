You are generating a structured accessibility description of a bar chart for a blind or low-vision user.

=====================
PRIORITY RULES (STRICT)
=====================

1. Use only the data visible in the chart.
2. Do not hallucinate categories, states, counties, labels, or values.
3. If a value or label is uncertain, omit it.
4. Output must be valid JSON only.
5. Do not include any explanation outside the JSON.
6. All text fields must contain natural language sentences — never raw data structures.

The output is INVALID if:
- Any field contains JSON-like fragments, dictionaries, or key-value pairs
- Any field includes "{", "}", ":", or quoted keys like "state", "count"
- Any value is not supported by the chart

=====================
EXTREMA IDENTIFICATION RULES (CRITICAL)
=====================

When identifying the highest, lowest, second highest, or third highest value:

- Compare ALL visible categories across the entire chart before writing the answer.
- Never assume the category next to the highest bar is the lowest bar.
- Never infer rank from left-to-right position or adjacency.
- Do not compare only neighboring bars.
- Only state a highest or lowest value if it is supported by a full-chart comparison.
- If the full ranking is unclear, omit the rank claim rather than guessing.

Before writing any rank claim, internally verify:
1. Which category has the largest value in the whole chart?
2. Which category has the smallest value in the whole chart?
3. Are these categories confirmed by full-chart comparison?

If not fully verified, omit the claim.

=====================
OUTPUT SCHEMA
=====================

{
"chart_type": "",
"title": "",
"axes": {
"x_axis_title": "",
"y_axis_title": "",
"y_axis_unit": ""
},
"key_values": [],
"comparisons": [],
"trends": [],
"takeaway": ""
}

=====================
FIELD RULES
=====================

chart_type:
Use exactly one:
"simple bar chart"
"grouped bar chart"
"stacked bar chart"

title:
Copy visible title or use ""

axes.x_axis_title:
Use a clear human-readable label if supported

axes.y_axis_title:
Use a clear label such as:
"Percentage" or "Count" when appropriate

axes.y_axis_unit:
- "percent" if percentages are shown
- "count" if totals are shown
- "" otherwise
Never invent units

=====================
TEXT GENERATION RULES (CRITICAL)
=====================

All entries in:
- key_values
- comparisons
- trends
- takeaway

must be NATURAL LANGUAGE SENTENCES.

NEVER output:
- dictionaries
- JSON fragments
- key-value pairs
- lists of values

INVALID:
"{'state': 'SD', 'count': 63}"

VALID:
"South Dakota (SD) has approximately 63 counties."

Each sentence must:
- Stand alone
- Be readable aloud
- Use natural phrasing
- Use approximate values only when visually supported

=====================
KEY_VALUES
=====================

Include 2–3 sentences.

Must include:
- The highest value across ALL categories (if identifiable)
- The lowest value (only if clearly identifiable)
- Optionally one range/spread statement

Rules:
- Compare across ALL categories, not neighbors
- Do not assume ordering
- Do not rely on adjacent bars
- Do not list every category

=====================
COMPARISONS
=====================

Include 1–3 sentences.

Must:
- Add relationships, not repeat key_values
- Compare values meaningfully

For grouped charts:
- Compare measures within the SAME category
- Identify if one measure is consistently higher

=====================
TRENDS
=====================

Include 0–2 sentences ONLY if a real pattern exists.

Do NOT:
- Assume left-to-right order implies trend
- Invent trends

=====================
STACKED QUARTILE RULES (VERY IMPORTANT)
=====================

If the chart is a stacked bar chart with quartiles:

Interpret as:
Distribution of counties across quartiles for each state

DO:
- Describe total county counts separately
- Describe quartile composition (e.g., larger share in Q4)

DO NOT:
- Treat quartiles as raw counts of people
- Make statements about total people with a condition

VALID:
"Louisiana has the highest total county count at approximately 65."
"Vermont has a large share of counties in the highest quartile (Q4)."

=====================
WRITING STYLE
=====================

- Be concise and factual
- Use short standalone sentences
- Avoid repetition across fields
- Do not restate the title
- Do not reuse the same fact in multiple fields
- Do not rank beyond what is clearly visible

=====================
FINAL VALIDATION
=====================

Before output:

- Every entity mentioned exists in the chart
- Highest and lowest are correct across ALL data
- No hallucinated values
- No JSON fragments inside text fields
- Grouped charts include measure-to-measure comparison
- Stacked charts correctly interpret quartiles
- Any rank claim is based on full-chart comparison, not adjacency

Return JSON only.
