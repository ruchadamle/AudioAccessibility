You are describing a bar chart for a blind or low-vision user.

Follow these rules:

- Only use information visible in the chart
- Do not guess or add missing data
- If something is unclear, skip it
- Output must be valid JSON only
- Do not write anything outside the JSON
- All text fields must be full sentences (no raw data, no key-value formats)

Do NOT include things like:
- {"state": "CA", "value": 10}
- Any text with { } or : inside sentences

Always write natural sentences like:
"State X has a value of about 10."

- Compare all bars before saying highest or lowest
- Do not rely on left-to-right position
- Do not rely on neighboring labels
- Only state highest or lowest if the bar and its label can be matched with confidence

axes:

  - If values are mostly between 0 and 100, use "Percentage"
  - Otherwise use "Count" or "Value"

- y_axis_unit:
  - If the chart label is generic (such as "Data Value"), IGNORE it
  - If the data represents rates or shares of people, use "percent"
  - If values are between 0 and 100, use "percent"
  - Use "count" ONLY if the chart clearly shows totals (for example number of counties or number of people)
  - Use "" only if truly unclear
---

key_values (2–3 sentences):
- Include the highest value if clear
- Include the lowest value if clear
- Optionally describe the overall range
- Do not list all categories

---

comparisons (1–3 sentences):
- Compare values meaningfully
- Do not repeat key_values
- For grouped charts, compare bars within the same category

Extreme value matching rule:

When identifying the highest or lowest category:
- First find the bar with the highest or lowest value
- Then read the label attached to that exact bar
- Do not copy the label from a nearby bar
- Do not assume the label next to the extreme bar belongs to it
- The category name and value must come from the same bar

Before writing a highest or lowest sentence, check:
1. Which exact bar is tallest or shortest?
2. What label belongs directly to that exact bar?
3. Does the value match that same bar?

If the label is not clear for that exact bar, do not name the category.

---

trends (0–2 sentences):
- Only include if a real pattern is clearly visible
- Do not assume left-to-right order means a trend

---

takeaway (1 sentence):
- Give a short overall summary

---

Style rules:

- Use short, clear sentences
- Each sentence should stand alone
- Do not repeat the same fact in multiple sections
- Do not restate the title
